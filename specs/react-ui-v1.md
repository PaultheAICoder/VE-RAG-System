## 1. Overview

### 1.1 Purpose

Replace the Gradio UI with a modern React application to provide:
- Better reactivity and user experience
- Consistent styling with AI Ready PDX brand
- Role-based views for different user types
- Real-time chat with WebSocket streaming
- Professional admin dashboard

### 1.2 Goals

- **Component Reuse**: Build a reusable component library
- **Styling Control**: Full Tailwind CSS customization with design tokens
- **Modern UX**: Responsive layout, dark mode, keyboard shortcuts
- **Maintainability**: TypeScript, clean architecture, separated concerns

### 1.3 Non-Goals (Rev 2)

- Mobile responsive design
- Embeddable chat widget for company intranet
- PWA/offline support

## 3. User Roles & Permissions

### 3.1 Role Definitions

| Role | Description |
|------|-------------|
| **System Admin** | Full access - IT/infrastructure owner |
| **Customer Admin** | Manages users, documents, tags for organization |
| **User** | Consumes RAG chat, views permitted documents |

### 3.2 View Access Matrix

| View | System Admin | Customer Admin | User |
|------|:---:|:---:|:---:|
| Chat | âœ“ | âœ“ | âœ“ |
| Documents | âœ“ | âœ“ | âœ“ (filtered by tags) |
| Tags | âœ“ | âœ“ | âœ— |
| Users | âœ“ | âœ“ | âœ— |
| Settings | âœ“ | âœ— | âœ— |
| Health | âœ“ | âœ— | âœ— |

### 3.3 Permission Details

- **Customer Admin** can:
  - Create, edit, deactivate users (including other Customer Admins)
  - Assign any tag to users
  - Upload, edit, delete documents
  - Create, edit, delete tags

- **User** can:
  - Chat with RAG system
  - View documents they have tag access to
  - Cannot see admin navigation items

### 6.2 WebSocket Chat Streaming

```typescript
// WebSocket connection for real-time chat
const ws = new WebSocket(`wss://${host}/ws/chat?token=${token}`);

ws.onmessage = (event) => {
  const data = JSON.parse(event.data);
  if (data.type === 'token') {
    // Append token to current response
    appendToken(data.content);
  } else if (data.type === 'done') {
    // Response complete
    finalizeResponse(data.sources, data.confidence);
  }
};
```
